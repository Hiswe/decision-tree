## requirements

- [node](http://nodejs.org/download/) >= 10.15.1
- [yarn](https://yarnpkg.com/lang/en/) >= 1.13.0

## commands

### building dependencies

```sh
yarn install
```

### launching for test

```sh
yarn start
```

application will be accessible at `http://localhost:8010/`
